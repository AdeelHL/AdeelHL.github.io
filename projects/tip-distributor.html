Tip Distributor

A web app for fairly splitting and distributing tips across staff. Clean, easy-to-use UI with tailored logic for personal, practical use.

Live Demo → GitHub

About the Project

The Tip Distributor is a web application designed to simplify and automate how tips are divided among employees in my workplace. In many small businesses like ours, tip distribution is often calculated on paper, which can be time-consuming and prone to mistakes. This application allows managers to input tip amounts and work hours, and then automatically calculates and distributes tips based on predefined rules.

The goal of the project was to create a simple and reliable tool that improves transparency and efficiency in tip management. I designed the logic to calculate each employee’s share and built the interface to let users easily enter and review data. This project helped me practice building frontend functionality, handling user input, and implementing calculation logic to solve a real-world problem.

At the time, I had no idea how to host this properly, nor did I think it was entirely necessary, so I opted for a simple .bat file to execute this locally
