AI Threat Hunter

A security-focused AI chatbot that analyzes Wazuh logs using RAG. Queries a vectorized log store via LLaMA3 to surface threats and anomalies in real time.

🔒 Repo is Private

About the Project

AI Threat Hunter is a RAG-based security chatbot built on top of Wazuh log data. It ingests logs from Elasticsearch, chunks and vectorizes them using FAISS and HuggingFace embeddings, then lets analysts query them in plain English through a WebSocket-powered chat interface.

The backend runs on FastAPI with LangChain's ConversationalRetrievalChain wired to a locally hosted LLaMA3 model via Ollama — no external API calls, fully self-hosted. Analysts can reload the vector store, adjust the log date range, and run stats queries on the fly using tailored commands.

Deployment is handled via a Python daemon mode, allowing the server to run as a background process on the Wazuh host itself. Authentication is enforced on all routes using HTTP Basic Auth, keeping the interface locked down to authorized analysts only.
