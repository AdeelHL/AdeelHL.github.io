MCP Log Forwarder

A self-hosted log routing server with a web UI for managing forwarding destinations. Supports multiple destination types including Elasticsearch, configured dynamically through a JSON-backed interface.

GitHub

About the Project

MCP Log Forwarder is a lightweight log routing server built with FastAPI and Jinja2. It provides a web interface for registering and managing forwarding destinations without touching config files manually.

Destinations are stored in a JSON config file and loaded at runtime. Through the UI, you can add new destinations by specifying the type, URL, credentials, and index name. The server handles routing incoming log data to the correct destination based on the config.

The project is designed to run self-hosted in internal network environments where log data shouldn't leave the local infrastructure, making it a practical companion to tools like Wazuh or other SIEM pipelines.
